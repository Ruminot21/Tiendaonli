Uso en teléfonos

Acceso para meseros
Modo móvil ya incluido
Este archivo se adapta a celulares y tablets. Los meseros pueden entrar desde su teléfono si publicas el HTML en un hosting o servidor local del restaurante.
Importante
En un solo archivo HTML con localStorage, cada dispositivo guarda sus propios datos. Para que administrador, cocina y meseros compartan información en tiempo real entre varios teléfonos se necesita backend o base de datos en red.
Flujo incluido
Cuando cocina marca un pedido como Listo, el sistema crea una notificación interna para el mesero que tomó ese pedido.
